# EduConnect – Student Wellbeing Console System (C#)

A role-based student wellbeing and academic support console system. Students can record wellbeing updates, supervisors can review them, and senior tutors can oversee records and engagement.

## Features
• Student, Supervisor, and Senior Tutor roles  
• Submit, review, and track wellbeing logs  
• Data persistence using JSON storage  
• Menu-driven console interface and input validation  

## Technologies Used
• C# (.NET)  
• JSON storage  
• GitHub Actions (CI)

## Key Learning Outcomes
• OOP architecture and modular program design  
• File handling and data serialization in C#  
• Console application workflow and state management  
• CI workflow to validate builds

## Run Instructions
```bash
dotnet build
dotnet run
